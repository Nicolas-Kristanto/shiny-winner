# 🌟 Shiny-Winner

**Shiny-Winner** adalah kumpulan proyek permainan kreatif yang dibangun menggunakan platform **Scratch**. Koleksi ini mendemonstrasikan pemahaman tentang algoritma dasar, *event handling*, dan desain interaksi visual yang menarik.

- **Variables & Scoring:** Sistem skor dan level yang dinamis.
